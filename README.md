# ASTE Project — ModernBERT + GCN

## Quick start (Ubuntu + NVIDIA GPU)

```bash
# 1. Install everything
bash setup.sh

# 2. Merge the two datasets
python merge_data.py

# 3. (Optional) Verify data loader
python data_loader.py

# 4. Train
python train.py \
  --data merged_train.jsonl \
  --epochs 10 \
  --batch_size 16 \
  --lr 2e-5

# 5. Predict on test set (once released)
python predict.py \
  --test test_data.jsonl \
  --checkpoint checkpoints/best_model.pt \
  --output predictions.jsonl
```

## File structure

```
.
├── setup.sh            # Install all dependencies
├── merge_data.py       # Merge laptop + restaurant → merged_train.jsonl
├── data_loader.py      # Dataset, BIO labelling, dep graph, DataLoader
├── model.py            # ModernBERT + GCN + Span head + VA head
├── train.py            # Training loop, evaluation, checkpointing
├── predict.py          # Inference on test set → predictions.jsonl
├── report.md           # Model description report (fill results after training)
├── merged_train.jsonl  # Combined training data (generated by merge_data.py)
└── checkpoints/
    ├── best_model.pt   # Best checkpoint by val span F1
    └── history.json    # Per-epoch metrics
```

## Useful training flags

| Flag | Default | Description |
|---|---|---|
| `--epochs` | 10 | Number of training epochs |
| `--batch_size` | 16 | Batch size |
| `--lr` | 2e-5 | Encoder learning rate (heads get 10×) |
| `--gcn_layers` | 2 | Number of GCN layers |
| `--lambda_va` | 0.5 | Weight of VA loss relative to span loss |
| `--no_dep_graph` | off | Skip spaCy parsing (faster, lower quality) |
| `--val_split` | 0.1 | Fraction of data held out for validation |

## Output format

```json
{"ID": "R001", "Triplet": [{"Aspect": "Indian food", "Opinion": "average to good", "VA": "6.75#6.38"}]}
```
